tar xzvf alcon2019.tar.gz

説明

  • csvファイル
    • dataset/test/annotations.csv: 提出するファイル．（空です）
    • dataset/train/annotations.csv: 学習ファイルのラベルリスト
    1行目はヘッダです． それ以降の行は，1つの画像のIDと3文字（上，中，下それぞれ）のラベル三つです． 以下は例です． 
    ID,Unicode1,Unicode2,Unicode3
0,U+3084,U+308C,U+3068
1,U+306E,U+304B,U+306A
2,U+304B,U+306A,U+3057
...
    ラベルはUnicode code pointsで， 範囲はU+3042（ひらがなの「あ」）から U+3093（ひらがなの「ん」）までです． code pointsのリストは次のURLなどを参照してください． " UTF-8 encoding table and Unicode characters; page with code points U+3040 to U+30BF"
  • 画像ファイル
    • dataset/test/imgs/: 16387枚（0から16386まで）のテストセットの画像があるフォルダ．
    • dataset/train/imgs/: 119997枚（0から119996まで）の学習セットの画像があるフォルダ．
    ファイル名の数字は画像のIDです． 例えば0.jpgのIDは0，100.jpgのIDは100です．
  • 追加画像ファイル
    • train_kana/ フォルダには学習用の追加画像があります． 各画像には1つの文字です． アルゴリズムの開発のためにしようして構いません．
  • サンプルコード
    • main.pyとsrcs/ フォルダ中のファイルはpythonのサンプルコードです． これらを使っても構いません．
