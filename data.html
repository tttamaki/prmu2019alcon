Dataset

Download **(larger than 2.5GB)**

wget http://www.iic.ecei.tohoku.ac.jp/~tomo/alcon2019.tar.gz
tar xzvf alcon2019.tar.gz

Contents

  • csv files
    • dataset/test/annotations.csv: a blank file that you will submit.
    • dataset/train/annotations.csv: a label list for train files.
    The first line is the header. Following lines are ID and three labels for the three characters (top, middle, and bottom) in a single image. Here is an example. 
    ID,Unicode1,Unicode2,Unicode3
0,U+3084,U+308C,U+3068
1,U+306E,U+304B,U+306A
2,U+304B,U+306A,U+3057
...
    Labels are Unicode code points ranging from U+3042 (HIRAGANA LETTER A) to U+3093 (HIRAGANA LETTER N). See the list of code points at " UTF-8 encoding table and Unicode characters; page with code points U+3040 to U+30BF"
  • image files
    • dataset/test/imgs/: folder containing the test set of 16387 test image file (numbered from 0 to 16386).
    • dataset/train/imgs/: folder containing the training set of 119997 training image file (numbered from 0 to 119996)
    The number in the file name stands for the ID of the image. For example, 0.jpg is ID 0, 100.jpg is ID 100, and so on.
  • extra image files
    • train_kana/ folder have additional training images. Here individual characters are separated. These images may or may not be used for developing algorithms.
  • starter kit
    • main.py and files in srcs/ folder are python scripts for starting the development. These files may or may not be used.
